    Low-level wrappers for the GMT C API¶

    The GMT C shared library (libgmt) is accessed using ctypes. The gmt.clib package offers the LibGMT class that wraps the C shared library with a pythonic interface. Most interactions with libgmt are done through this class.

    clib.LibGMT([libname]) Load and access the GMT shared library (libgmt).
